📐 配置プリセット

📘 Cheer Choreo 使い方ガイド

🎵 音源を入れる

🎵 音源 をタップ → 端末から音楽ファイル（mp3など）を選ぶ
BPM を曲のテンポに合わせて入力（自動検出は「⚙ 詳細」から）

👤 人の位置を変える（3通り）

① ドラッグ：人を指で押さえて動かす
② タップ移動：人をタップ → 📍 タップで移動 → 行き先をタップ
③ 微調整：人をタップ → ← ↑ → ↓ ボタンで2%ずつ動かす

⬆️ 高さを変える（地上 / 肩 / エクステンション）

人をタップ → 操作カードの 地上 肩 エクステンション ボタン
または人を ダブルタップ で順番に切り替え

🤸 役割を変える（ベース・トップ・スポッター・バックスポット）

人をタップ → 操作カードの 役割 ドロップダウン

🏗 スタンツを組む

1. ベース役を2人以上、トップ役を1人以上用意
2. トップ役の人をタップ → 操作カードの ＋ スタンツ作成
3. 自動でベース2人＋スポッター＋バックスポットが割り当てられる
4. 微調整は「⚙ → スタンツタブ」で各役割をドロップダウンから選び直し
5. ボディポジション（リバティ・ヒールストレッチ等）も同タブで設定
💡 スタンツ参加メンバーには 黄色いバッジ（T/B/Sp/Bk）が表示されます

🔗 スタンツのまま移動

スタンツ参加メンバーをタップ → 🔗 連動移動 ON
そのままドラッグ → スタンツのメンバー全員が一緒に動く
💡 例：トップを動かす → ベース2人＋スポッターも追従

🎬 隊形の切替・追加

切替：画面下のタイムラインの 隊形カード をタップ
追加：ヘッダーの + 隊形（現在の隊形がコピーされる）

⏯ 1カウントずつ動きを決める

◀ ▶ で1カウントずつ進める／戻す（スクラブモード）
スクラブ中にメンバーを タップ／ドラッグ → 自動でその位置に分割され、新しい隊形を編集できます
明示的に分割したい場合は ✂ ここで分割 ボタン
スクラブを抜けるには 終了 または タイムラインの隊形をタップ

🧲 スナップ（マットの線・中央に吸い付き）

ヘッダーの 🧲 スナップ（既定 ON）：
メンバーをドラッグ／タップ移動すると、マットの境界線や各マットの中央付近で「カチ」と吸い付きます
💡 自由配置したいときはボタンを再タップで OFF にできます

▶️ 再生

ヘッダーの ▶ 再生 → 隊形遷移と音源を同期再生
速度・ループは「⚙ 詳細」を開いて設定

👁 ビュー切替

真上：マット上の配置を確認・編集
正面：観客から見た人型・スタンツポーズを確認

🏟 マット規格

USA(9)：9マット（5本目が中央）
JCA(7)：7マット（4本目が中央）

📤 共有・配布

ヘッダーの 📤 共有 をタップすると 4つの方法が選べます：
① URL コピー：振付データを丸ごと URL に埋め込んでコピー。誰にでも送れる
② LINE で送る：URL を LINE のシェアダイアログで送信
③ JSON 書き出し：ファイルでバックアップ／別端末への持ち運び
④ JSON 読み込み：保存したファイルから振付を復元

❓ 困った時

人が動かない → 画面を一度再読み込み（更新ボタン）
操作カードが画面を圧迫 → カードの ✕ 解除
スクラブから戻る → 終了 ボタン
音源が再生されない → 画面を一度タップしてから ▶ 再生（iOS制限）
保存データを消したい → 📤 共有 →「保存データを削除してサンプルに戻す」
💾 自動保存対応：編集内容はこの端末（ブラウザ）に自動保存されます。リロードしても消えません。
※ 別端末で続きを作るときは 📤 共有 から URL or JSON で持ち運んでください。

📤 共有・データ管理

🔗 リンクで共有

URL に振付の全データが入っています。LINE・メール・Slack 等で誰にでも送れます。受け取った人は URL を開くだけで同じ振付を見られます。

💾 ファイルで保存・受け渡し

この振付をバックアップ／別端末への持ち運び用に保存します。ダウンロードフォルダに cheer-choreo-YYYY-MM-DD-HH-mm.json が保存されます。
⚠️ 現在編集中の振付は上書きされます。

🗑 端末の保存データを消去

この端末のブラウザに保存された振付を完全に消去し、初期サンプル（3 隊形）に戻します。
